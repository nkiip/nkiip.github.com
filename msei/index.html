:relaxed:服务器使用须知

:cat: 服务器是实验室每个同学都要用到的，使用过程中难免会有一些小问题，现总结如下：

  • :eyes:每台服务器具体端口号都用标签贴在服务器上，可自行查看
  • :cop:现实验室共有大小服务器6台，其中3台新的（包括2台GPU服务器及1台戴尔塔式服务器）处于一直运行状态，10004为Windows服务器，10005为Win Server 2012服务器，20005为CentOS服务器；另外3台也可开机正常使用，10000,10001为Windows服务器，20004为CentOS服务器
  • :thumbsup:重启服务器时一定要在群里询问有没有同学在跑实验，或者通过任务管理器查看一下，然后再进行重启
  • :joy:新安装的软件不要直接安装在C盘，相信大家都有这个意识了，一般服务器软件都会装在D盘下program files或x86文件夹下
  • :minidisc:不要轻易重装系统，因为重装系统、软件以及进行新系统的配置使得大家都能用是很费时费力的
  • :feet:服务器清灰有专用的吸尘工具，硬盘及内存条更换请自行百度
  • :kissing_smiling_eyes:服务器是实验室重要的公共资源，希望大家好好爱护保养，然后才是Enjoy it:recycle:
